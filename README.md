# Savings Goals

A [Treeline](https://github.com/zack-schrag/treeline-money) plugin for tracking savings goals.

Track progress toward financial goals like:
- Emergency fund
- House down payment
- Vacation fund
- New car
- Any custom goal

## Installation

### From Community Plugins (Recommended)

1. Open Treeline
2. Go to Settings > Plugins > Community Plugins
3. Find "Savings Goals" and click Install
4. Restart Treeline

### Manual Installation

```bash
tl plugin install https://github.com/zack-schrag/treeline-goals
# Restart Treeline
```

## Development

```bash
git clone https://github.com/zack-schrag/treeline-goals
cd treeline-goals
npm install
npm run build
tl plugin install .
```
